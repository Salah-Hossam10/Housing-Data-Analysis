# Housing Data Analysis and Sale Price Prediction

## Project Overview

This project is a Data Scientist take-home assignment focused on analyzing housing data and building a predictive model for house sale prices.

The main goal is to understand which property features have the strongest relationship with `SalePrice`, create clear visual insights, and build a machine learning model that can estimate house prices from historical data.

The work includes exploratory data analysis, data cleaning, preprocessing, visualization, predictive modeling, model comparison, model evaluation, and a simple real-estate recommendation system concept.

---

## Dataset

The dataset contains housing records with property-related features such as overall quality, living area, garage area, basement size, year built, neighborhood, number of bathrooms, sale condition, and sale price.

The target column is `SalePrice`.

The dataset used in this project contains 900 rows and 82 columns.
